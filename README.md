# FastLogs

Кросс-движковый диагностический инструмент: из запущенной игры одним жестом отправить рантайм-логи + полный диагностический срез устройства (+ опц. скриншот) на свой сервер и получить **короткую ссылку** для быстрого просмотра в браузере. Ссылки автоматически складываются в каталог `Server -> Project -> version -> Log`.

Работает на всех платформах (WebGL/веб-порталы, iOS, Android, Standalone) и на нескольких движках (**Unity**, **GameMaker**). На консолях полностью вырезается из релизных билдов (без конфликтов на сертификации).

> **Лицензия:** MIT (см. [`LICENSE`](LICENSE)). Сервер задуман переиспользуемым - лицензию можно сменить под свои нужды, дефолты нейтральны (`BASE_URL=http://localhost:8787`).

## Структура репозитория

| Каталог | Что внутри |
|---------|-----------|
| [`CONTRACT.md`](CONTRACT.md) | **Источник правды** - единый JSON-контракт API для всех клиентов |
| [`server/`](server/) | Сервер (Node + SQLite за nginx): ingest, вьюер, каталог `/browse`, sinks, ретеншн, sweeper, деплой. Разворачивается через Docker (`docker compose up -d`) или systemd (см. `server/deploy/INSTALL.md`) |
| `unity/` | UPM-пакет клиента для Unity (C#); ставится по git-URL _(планируется)_ |
| `gamemaker/` | GML-пакет клиента для GameMaker (+ собранный `.yymps`) _(планируется)_ |
| `docs/` | Материалы для документации команды _(планируется)_ |

## Быстрый старт (сервер)

```bash
cd server
cp .env.example .env        # задать BASE_URL, ADMIN_TOKEN, VIEWER_TOKEN, IP_SALT
docker compose up -d        # либо: npm install && npm start
npm run add-app -- mygame "My Game" 30
```

## Использование

В деве: жест/хоткей -> открывается оверлей (счётчики E/W/L, опц. тоггл «Скриншот») -> «Отправить» -> ссылка (копируется в буфер + QR). По ссылке - вьюер с фильтрами/поиском. Командный каталог `/browse` (под viewer-токеном) листает все логи по проектам и версиям. Записи живут настраиваемый срок (по умолчанию 30 дней), «закреплённые» (pin) не удаляются.

## Безопасность

- Логи/скриншоты могут содержать персональные данные -> непубличный доступ по непредсказуемой ссылке, ограниченный ретеншн, каталог под командной авторизацией.
- Секреты (ключи деплоя, токены) - никогда в репозиторий (см. `.gitignore`).
